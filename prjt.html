Projects

The following projects reflect my academic journey and practical experience in computing. Each project demonstrates my ability to apply theoretical knowledge into structured, efficient, and problem-oriented solutions. Through these implementations, I have strengthened my programming skills, analytical thinking, and understanding of core computer science concepts.

01.Maze Runner Project

The Maze Runner Game is a fully functional console-based application developed in C. The project implements structured programming principles and file handling mechanisms to dynamically load game elements such as walls, poles, stairs, and flags from external text files. The game logic includes player movement control, obstacle interaction, random bonus generation, and win-condition validation. This project demonstrates my understanding of data structures, control flow management, and modular program design while emphasizing algorithmic thinking and efficient memory usage.

02.Binary Search Tree (BST) Implementation Project

This project focuses on implementing a Binary Search Tree data structure in C. It includes core operations such as node creation, insertion, deletion, searching, and traversal (in-order, pre-order, and post-order). The implementation strictly follows BST properties to maintain ordered data storage. Through this project, I strengthened my understanding of recursive algorithms, pointer manipulation, dynamic memory allocation, and tree-based data structures.

01.SRS Project

The Smart Public Library System - Business Requirements project focuses on analyzing and documenting the functional and non-functional requirements of a modern digital library management system. This Software Requirements Specification (SRS) defines system objectives, stakeholder needs, operational constraints, and performance expectations. The project emphasizes requirement elicitation, structured documentation, use case identification, and system modeling to ensure clarity, scalability, and maintainability in software development.
